* Interesting things learned in this Project
** Installing Packages
#+begin_src bash
npm init
npm i --save-dev sass gulp-sass
npm i --save-dev gulp
npm i --save-dev gulp-imagemin@7.1.0
npm i --save-dev bootstrap
#+end_src
